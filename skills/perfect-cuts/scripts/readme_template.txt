PERFECT CUTS — YOUR EDITED VIDEO PACKAGE
=========================================
Skill Created by Vic Laranja. Get more Free Creator Skills at https://skool.com/vic

Your raw clip was cut down automatically: retakes removed, false starts
caught, dead air stripped. Every file in this folder is the SAME edit,
in a different format. Use the one that matches your workflow:

1 WATCH - final video (C).mp4
   The finished cut as a normal video. Watch it, post it, or import it
   into CapCut or any editor as a regular clip.

2 EDIT - Premiere + Resolve (C).xml
   The cut as an editable timeline (links to your ORIGINAL clip).
   - Premiere Pro:     File > Import > pick this file
   - DaVinci Resolve:  File > Import Timeline > pick this file
   Every cut point stays adjustable.

3 REVIVE - cut decisions (C).csv
   Open in Excel/Numbers/Google Sheets. Every line of your raw clip,
   whether it was KEPT or CUT, and why. If a line was cut that you
   wanted, tell your assistant "revive block [number]" and the package
   gets re-exported with it back in.

4 OPEN IN REMOTION - Mac (C).command
5 OPEN IN REMOTION - Windows (C).bat
   Double-click to open the edit in Remotion Studio (free, code-based
   video tool). Needs Node.js installed (the launcher tells you if it's
   missing). First run installs Remotion - takes a few minutes.

6 CAPTIONS (C).srt
   Subtitles timed to the EDITED video. Pairs with the MP4 in CapCut,
   YouTube, or any caption-friendly tool.

7 AVID + LEGACY (C).edl
   Old-school timeline format for Avid and other pro apps.

cut data (C).json
   The raw cut points (machine-readable). Powers the Remotion launcher
   and any automation. Leave it in this folder.

GOOD TO KNOW
------------
- The XML, EDL, and Remotion launcher all point at your ORIGINAL raw
  clip. If you move or rename that clip, re-link it in your editor (or
  drop the raw clip into this folder for the Remotion launcher).
- The MP4 is the only file that works standalone forever.
